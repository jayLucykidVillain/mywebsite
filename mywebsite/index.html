A short journey blog

The road goes ever on and on...

I've always liked solving problems and using my head, so it makes sense that I'd be attracted to coding and web development. I learned a little bit of HTML back when I was in high school, but that was ages ago. In just a few short weeks with the Correlation One program, I've learned how to make a basic web page that includes forms, tables, links to other sites, images, videos, etc. I understand how useful and essential CSS can be in making the appearance of a site go from 0 to 100 real fast.

While overall I have found everything interesting, I find the availability of free resources for developers phenomenal. I particularly like using colorhunt as there are a variety of color palettes to use and a useful feature is that a certain color's hex code is instantly copied to the clipboard when clicked on! I'm also finding myself using the "view page source" feature on sites I visit often. I might not understand everything yet, but there are some things that stand out.
